Codigo github-bash (terminal)
ls - lista os arquivos dentro da pasta
ls -all  - mostra tudo que existe na pasta
clear - limpar terminal
cd + pasta onde quer ir - muda de diretorio para onde quer ir
cd .. - volta diretorio anterior
mkdir + nome - cria pasta
touch + nome - cria arquivo
rm + nome do arquivo - apaga ele
rm -r + nome da pasta - apaga pasta

git init - iniciar um repositorio no git
git add + nome do arquivo - para adicionar no repositorio
git add -A - adiciona todos items para o repositorio
git add -U - só da update nos itens que eu mudei
git commit -m "nome que quero dar" - envia para o git hub os arquivos.
